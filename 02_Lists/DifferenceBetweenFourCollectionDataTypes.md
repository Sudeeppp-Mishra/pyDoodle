# Python Collections Comparison

| Type       | Ordered | Mutable | Allows Duplicates | Indexed / Keyed |
|------------|---------|---------|-------------------|-----------------|
| **List**   | ✅ Yes  | ✅ Yes  | ✅ Yes            | ✅ Indexed      |
| **Tuple**  | ✅ Yes  | ❌ No   | ✅ Yes            | ✅ Indexed      |
| **Set**    | ❌ No   | ✅ Yes  | ❌ No             | ❌ No indexing  |
| **Dict**   | ✅ Yes  | ✅ Yes  | ❌ Keys: No       | ✅ Key-based    |
